# 搜索功能

搜索功能的实现设计:

- 前段界面输入搜索关键字, 传送到对应view中
- 在对应的view中进行数据库关键字搜索
- 这里搜索可以只对文章名搜索或者全文搜索

在my_blog/article/views.py中添加查询逻辑

```
def blog_search(request):
    if 's' in request.GET:
        s = request.GET['s']
        if not s:
            return render(request,'home.html')
        else:
            post_list = Article.objects.filter(title__icontains = s)
            if len(post_list) == 0 :
                return render(request,'archives.html', {'post_list' : post_list,
                                                    'error' : True})
            else :
                return render(request,'archives.html', {'post_list' : post_list,
                                                    'error' : False})
    return redirect('/')
```

这里为了简单起见, 直接对`archives.html`进行修改, 使其符合查询逻辑

```

添加了if判断逻辑, 然后还需要修改`views中的archives`

```
def archives(request) :
    try:
        post_list = Article.objects.all()
    except Article.DoesNotExist :
        raise Http404
    return render(request, 'archives.html', {'post_list' : post_list, 
                                            'error' : False})
```

urlpatterns = patterns('',

```




#ReadMore功能

对于ReadMore的前段按钮界面设置早已经添加过了, 所以这里只需要进行简单的设置就好了

通过使用Django中内建的filter就可以速度实现

```
{{ value|truncatewords:2 }} #这里2表示要显示的单词数, 以后的会被截断, 不在显示
```

这里只需要修改my_blog/templates/home.html界面中的变量的过滤器

```
#将正文截断设置为10
 {{ post.content|custom_markdown|truncatewords_html:100 }}
```
